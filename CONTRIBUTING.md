# Contributing to OneFineStarstuff

First off, thank you for considering contributing to OneFineStarstuff! It's people like you that make this research ecosystem a great place to explore and learn.

## How Can I Contribute?

### Reporting Bugs
*   Check the [Issues](https://github.com/OneFineStarstuff/OneFineStarstuff/issues) to see if the bug has already been reported.
*   If not, open a new issue with a clear title and description, including steps to reproduce the problem.

### Suggesting Enhancements
*   Open an issue to discuss your proposed changes or new notebooks.

### Pull Requests
1.  Fork the repo and create your branch from `main`.
2.  Add your notebook or make your changes.
3.  Ensure your code follows the existing style and is well-commented.
4.  Open a Pull Request with a clear description of your changes.

## Coding Style
*   For notebooks, ensure all cells run correctly and unnecessary outputs are cleared before submitting.
*   Maintain clear documentation within the notebooks using Markdown cells.

## Community
*   Please adhere to our [Code of Conduct](CODE_OF_CONDUCT.md) in all your interactions.
